# Meus Desafios DIO

Este repositório foi criado para armazenar atividades, desafios e projetos desenvolvidos durante meus estudos na DIO.

## Objetivo

Organizar os arquivos produzidos ao longo dos cursos, bootcamps e desafios práticos, mantendo um registro da minha evolução e dos conteúdos aplicados durante a jornada de aprendizado.

## Conteúdo

Aqui serão adicionados arquivos relacionados a:

- Desafios de projeto;
- Exercícios práticos;
- Estudos sobre inteligência artificial;
- Atividades desenvolvidas durante os cursos da DIO.

## Sobre

Repositório de uso educacional, criado para fins de estudo e entrega de atividades da DIO.
